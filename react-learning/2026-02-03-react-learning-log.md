## 8.4, 폼 밸리데이션

* 폼 밸리데이션(form validation): 사용자가 입력한 값이 유효한지 확인하고, 올바르지 않은 경우 경고를 표시하거나 폼 제출을 막는 작업.

### 8.4.2, 커스텀 밸리데이션 로직 추가하기

* **커스텀 밸리데이션**(custom validation): 개발자가 직접 유효성 검사 로직을 작성해 폼을 검증.

### 8.4.3, 라이브러리 사용하기

* Formik 라이브러리
  - 형식
    ```tsx
    <Formik initialValues={} onSubmit={}></Formik>
    ```
  - Formik 컴포넌트 내부에서 <form> 요소 대신 Form 컴포넌트를 사용하면 다음과 같은 장점이 있음.
    * 폼 상태 자동 연결
    * 이벤트 핸들링 간소화
    * 접근성 지원
