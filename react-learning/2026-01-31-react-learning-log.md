## 5.4, 이미지 렌더링하기

* public 폴더 하위에 저장한 이미지는 정적(static) 리소스로 취급

- 번들링(bundling): 여러 파일(이미지, CSS, JS 등)을 하나의 최적화된 결과물 파일로 묶는 과정
- 트랜스파일(transpile): 최신 문법의 코드를 웹 브라우저가 이해할 수 있는 형태로 변환하는 과정

## 6.5, 로직 구현하기

* **로직**(logic)이란 프로그램이 수행해야 하는 작업의 순서와 조건을 정의한 일련의 과정을 의미.

* 계산기를 단순히 책을 보고 따라 치고, 그 다음 이해하는데도 너무나 어렵다. 과연 아이디어만 갖고 새로 만들 수 있을까 걱정이 되었다. 그런데 ChatGPT와 상담을 하니, 이런 것은 정상이라고 한다. 처음에 '이해 -> 구현'이 아니라 '구현 -> 이해 -> 다시 구현 -> 이해' 이런 식으로 머릿속에 저장된 패턴을 늘리고 반복하면, 실력이 자란다고 한다. 천천히 나아가보자. 그리고 지금은 어려워도 점점 실력이 늘면, 점차 쉬워지거나 또는 어려워도 해나갈 수 있다.
