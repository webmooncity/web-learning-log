# 14장, 비동기 데이터 통신과 처리 기법.

## 14.1, 데이터 통신의 기초 개념.

* **데이터 통신**: 디지털 환경에서 정보를 주고받는 과정.

- **통신 프로토콜**(communication protocol)
- **HTTP**(Hyper Text Trasfer Protocol)
- **HTTP 메서드**(HTTP method)
  - GET
  - POST
  - PUT
  - PATCH
  - DELETE

- **API**(Application Programming Interface)
  - **REST API**
    - **URI**
    - **엔드포인트**(endpoint)
    - **무상태성**(stateless)
  - **GraphQL**
    - **쿼리 언어**(query language)
    - **서브스크립션**(subscription)
