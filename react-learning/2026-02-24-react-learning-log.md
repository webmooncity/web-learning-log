## 13, Selfcheck.

조건
1. 도서 목록 페이지
  [x] 도서의 id와 제목을 나열합니다(<table> 태그 사용).
  [x] 도서 제목은 각 도서의 상세 페이지로 이동하는 링크입니다.

2. 도서 상세 페이지
  [x] 선택한 도서의 id, 제목, 저자, 출판 연도, 설명을 표시합니다(table 태그 사용).
  [x] URL에는 도서의 고유 식별자(id)가 포함되어야 합니다.
  [x] id에 따르나 상세 정보가 없는 경우 '도서의 상세 정보가 없습니다.'를 화면에 표시합니다.
    - 이것은 상세 구현 정답이랑 다름.

3. 라우팅
  [x] 도서 목록을 보여주는 경로는 /books이고, 상세 페이지는 /books/:id입니다.
  [x] 상세 페이지에서 도서 목록으로 이동하는 링크가 있습니다.
  [x] 루트 경로에 접속하면 /books 경로로 자동 이동됩니다.
  [x] 지정되지 않은 경로로 접속하면 '올바르지 않은 경로입니다.'를 화면에 표시합니다.
