## 5.4 가상 클래스 선택자 사용하기

* 가상 클래스 선택자 : 요소의 상태를 이용해 선택자를 지정하는 방법

- 링크 가상 클래스 선택자
- 동적 가상 클래스 선택자
- 입력 요소 가상 클래스 선택자
- 구조적 가상 클래스 선택자

## 5.5 다양한 선택자 조합하기

* 클래스 선택자나 아이디 선택자도 넓은 범위에서는 속성 선택자에 포함
* `*{}` : 전체 선택자
* `[속성=값]{}` : 기본 속성 선택자
  
* `h1{color:red;}`<br>
* `선택자{선언부}`, `{선언부}`는 `{속성:값;}`으로 구성됨

## CSS 필수 속성

## 6.1 CSS의 특징

- 기본 스타일 시트
- 적용 우선순위와 개별성, CSS(Cascading Style Sheets)
- 상속(inherit) : CSS 속성은 상속된다. 단, 상속되지 않는 속성도 있음. 그러나 외울 필요는 없음.
- 단위 : 절대 단위, 상대 단위
- 색상 표기법 : 키워드 표기법, RGB 색상 표기법, HEX 표기법
